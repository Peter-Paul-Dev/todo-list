The goal of this project is for me to create a web-based Todo-list application using HTML, CSS, JavaScript, and Webpack. 

My objective with this app is to demonstrate my understanding and sharpen my skills in the aforementioned languages as well create a functional and aesthetically pleasing application. In particular, this is meant to test my understanding and mastery of the JS language. This is where I aim make full use of factory-functions,  importing and exporting, closures, maybe class constructors, and more.

The finished product should be a fully functional Todo-list app with the basic functions such as creating Todos, deleting them, adding them to "project folders", making new projects, and marking them as complete.
